# Naming conventions

- Files: use kebab-case for file names under `src/` (`order-card.ui.tsx`).
- Folders: use kebab-case for feature folders, e.g. `menu-browse`.
- Components: use PascalCase for React components: `MyComponent`.
- Hooks: `useSomething` and placed in `hooks/` or `feature/model`.
- Types: suffix with `.types.ts` and use `PascalCase` for type names.
